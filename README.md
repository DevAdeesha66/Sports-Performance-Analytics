# Sports Performance Analytics – Manchester United (2024/25) ⚽

Analyzes Manchester United's 2024/25 EPL season using Python, Pandas, Matplotlib, and Seaborn.

## What it does 📊📈📉
- Cleans and structures match-level Excel data
- Engineers features: goal difference, match result, cumulative points
- Performs time-series analysis and moving average goal trends
- Visualizes goals scored vs. conceded and season performance patterns

## Tech Stack ⚙️
Python, Pandas, NumPy, Matplotlib, Seaborn

## How to Run 👨🏻‍💻⚙️
pip install pandas matplotlib seaborn openpyxl
python man_utd_analysis.py
